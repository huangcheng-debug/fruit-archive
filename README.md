# 🍎 水果知识档案

一个轻量级的水果百科 Web 应用，收录 50 种水果的详细信息，包括学名、分类、产地、季节、营养数据、趣味冷知识等。

## 特色

- 🔍 **智能搜索** — 支持按名称、学名、描述搜索，实时过滤
- 🏷️ **分类筛选** — 6 大水果分类（仁果类、热带水果、柑橘类、浆果类、核果类、瓜果类）
- 📊 **营养数据** — 每种水果提供热量、维生素C、膳食纤维等营养信息
- 💡 **趣味冷知识** — 每种水果配有有趣科学知识
- ⚙️ **管理后台** — 支持增删改查、导出/导入 JSON、数据重置
- 📱 **响应式设计** — 手机和电脑都能流畅浏览
- 🔌 **纯静态** — 无需服务器，双击 `index.html` 即可使用

## 快速开始

### 最简单的方式

双击打开 `fruit-archive/index.html` 即可使用！

### 使用本地服务器（推荐）

```bash
cd fruit-archive
python3 -m http.server 8080
```

然后在浏览器访问 `http://localhost:8080`

> 使用本地服务器可以避免浏览器跨域限制，确保数据 JSON 正常加载。

## 使用说明

### 浏览水果

- 首页展示所有水果的卡片
- 点击分类标签筛选类型
- 使用顶部的搜索框快速找到水果
- 点击卡片查看水果详细信息

### 管理水果

- 点击右上角「管理」按钮进入后台
- **添加水果**：点击「添加水果」按钮，填写表单
- **编辑水果**：点击表格中的「编辑」按钮
- **删除水果**：点击「删除」按钮
- **导出数据**：将当前数据导出为 JSON 文件备份
- **导入数据**：从 JSON 文件导入水果数据
- **重置数据**：恢复为默认的 50 种水果数据

### 快捷键

| 快捷键 | 说明 |
|--------|------|
| `Ctrl/Cmd + K` | 聚焦搜索框 |
| `Esc` | 清空搜索 / 关闭弹窗 |
| `Ctrl/Cmd + Enter` | 管理后台中保存表单 |

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- 零依赖，无框架
- localStorage 存储编辑数据
- CSS Grid 响应式布局
